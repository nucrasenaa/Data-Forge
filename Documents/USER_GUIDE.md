# 📖 User Guide

## 1. Connecting to a Database
Data Forge supports three primary modes of connection:
- **Manual Input**: Specify Server, Port, User, and Password.
- **Connection String**: Paste your full URI (e.g., `postgresql://user:pass@host:port/db`).
- **Secure Caching**: Use the "Securely cache credentials" option to save your connection for future use without re-entering passwords.

## 2. Browsing & Editing Data
- **Explorer**: Use the left sidebar to browse Databases, Tables, Views, and Procedures.
- **Inline Editing**: Double-click any cell in a Table View to edit it. Click the checkmark (✅) to save or (❌) to cancel.
- **Filtering**: Use the search bar at the top of a data tab to filter rows locally or via SQL.

## 3. Visual Experience
- **Theme Switching**: Toggle between **Light Mode** and **Dark Mode** using the button in the top-right corner of the Explorer screen or the bottom footer of a workspace.
- **Data Export**: Export your query results to **Excel, CSV, or JSON** using the export button on any data table.

## 4. Advanced Forge
- **Table Designer**: Create and modify tables visually without writing DDL.
- **ER Diagrams**: Generate entity relationship diagrams for any database or schema.
- **Performance Advisor**: Get AI-powered suggestions to optimize your queries.
